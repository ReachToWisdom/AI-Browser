# AI 브라우저

AI 모아 놓은 브라우저로, 종료시 트레이로 가고, 실제 종료 메뉴를 통해서 종료할것

## 기술 스택
- Go + WebView2 (Windows)
- 빌드: `go build -ldflags "-H windowsgui" -o AI_Browser.exe .`

## 주요 기능
- 탭별 AI 사이트 관리 (ChatGPT, Gemini, Claude 등)
- 시스템 트레이 (종료 시 트레이로 최소화)
- 자동 업데이트 확인 (GitHub Releases)
- 공유 세션 모드 (모든 탭 동일 세션)
- 팝업/새 창 → 시스템 기본 브라우저로 열기

## 알려진 이슈
- 네이버 메일 탭에서 메일 바로 읽기 작동 안됨 (팝업이 시스템 브라우저로 열림)

## 핵심 규칙

1. **SSOT**: 설정값/상수는 config 한 곳에 정의
2. **코드 주석 한국어**, 에러 처리 필수 (try-catch)
3. **Safe Parsing**: 모든 parseFloat에 기본값 필수
4. **파일 200줄 제한**

## 자기 검증

위 규칙을 위반하는 코드를 작성하려 할 때:
1. 즉시 중단하고 "⚠️ SPEC 위반: [항목]" 경고 출력
2. 사용자에게 승인 요청
3. 승인 없이 진행 금지
